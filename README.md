- これまでの講義などで作成した作品
  - デッサン、イラスト、ドット絵、背景、3Dキャラクターモデル、3D小物、3D背景など
  - 全てでなくても構いません
- 内定の状況

以上をWordやPowerpointにまとめるか、あるいはデータをまとめておいて、共有ドライブの提出フォルダーにコピーして提出できるようにまとめておいてください。
